# Qwix

A frontend library to initialize and use Qwik components instantly from custom HTML elements using directives.

## Installation

Install qwix in your projects by adding it to your dependencies:

- Using NPM

```shell
npm i -D qwix
```

- Using PNPM

```shell
pnpm i -D qwix
```

- Using Yarn

```shell
yarn add -D qwix
```

- Using Bun

```shell
bun i -D qwix
```
